Usage instructions:

1. Download all requirements in the requirements.txt folder
2. Start server on server device using 'python3 whatsapp_server.py'
3. Connect either locally (127.0.0.1) or remotely (IP of server device, configured in variable at the top of whatsapp.py)
4. Select mode of communication and enter username
5. Communicate locally and over the air!

Note: these instructions are the same for the encrypted version, but using 'whatsapp_server_encrypted.py' instead of 'whatsapp_server.py' and 'whatsapp_encrypted.py' instead of 'whatsapp.py'

Happy chatting!
